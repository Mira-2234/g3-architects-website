# 🏛️ G3 Architects - Responsive Architecture Website

Welcome to **G3 Architects**, a modern and responsive architecture website built using **HTML5 & CSS3**.  
This project showcases a clean, minimal and professional design for an architecture firm.

---

## 🚀 Live Demo
Live link: https://mira-2234.github.io/g3-architects-website/
---

## ✨ Features

### 🎯 Core Features
- Fully responsive layout (Mobile, Tablet, Desktop)
- Modern and clean UI design
- Smooth section-based layout
- Fast loading static website

### 🏗️ Website Sections
- Hero / Landing Section  
- About Company  
- Services Overview  
- Projects Showcase  
- Workflow Process  
- Contact Information

## 🧰 Technologies Used

- 🌐 HTML5 (Semantic Structure)
- 🎨 CSS3 (Modern Styling)
- 📦 Flexbox Layout
- 🧱 CSS Grid System
- 📱 Media Queries (Responsive Design)

## 📸 Project Overview

G3 Architects website is designed to represent a professional architecture firm.  
It focuses on simplicity, readability, and responsiveness across all devices.

The goal of this project is to practice real-world frontend UI development.

---

## 🎯 Purpose

- Improve frontend development skills  
- Practice responsive web design  
- Build portfolio-ready project  
- Learn layout structuring with Flexbox & Grid
